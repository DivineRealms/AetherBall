AetherBall is a Minecraft minigame plugin focused on fast, team-based matches with a physics-driven ball. It’s designed
to feel responsive and competitive without being overly complicated to set up or play.

## Features

* Team-based matches (red vs blue)
* Physics-based ball movement
* Match flow (lobby → starting → in-game → end)
* Scoreboards and goal messages
* Queue system and team system
* Economy integration (rewards and penalties)
* Rage quit penalty system

## Try it

You can test the plugin here:

**Server:** `divinerealms.org`

## Installation

1. Download the latest `.jar`
2. Place it into your `plugins/` folder
3. Start or restart your server
4. Configure via `settings.yml` and `messages.yml`

## Requirements

* Java 17+
* Spigot or Paper (Paper recommended)
